# 🎬 FilmFlux - Movie Discovery Platform

A modern, responsive movie discovery application built with React that allows users to explore and manage their favorite movies with a beautiful, intuitive interface.

## ✨ Features

### 🎯 Core Functionality
- **Movie Discovery**: Browse through a curated collection of latest movies
- **Search Functionality**: Find movies quickly with the advanced search bar
- **Favorites Management**: Add and manage your favorite movies
- **Responsive Design**: Perfect experience across all devices
- **Modern UI/UX**: Beautiful glass morphism design with smooth animations

### 🎨 Design Highlights
- **Glass Morphism**: Modern semi-transparent elements with blur effects
- **Gradient Backgrounds**: Stunning color schemes throughout the app
- **Smooth Animations**: Hover effects, transitions, and micro-interactions
- **Typography**: Professional fonts (Poppins & Inter) for optimal readability
- **Interactive Elements**: Engaging buttons, cards, and navigation

### 📱 User Experience
- **Intuitive Navigation**: Easy-to-use navbar with smooth transitions
- **Movie Cards**: Beautiful poster displays with hover effects
- **Newsletter Signup**: Stay updated with latest releases
- **Social Media Integration**: Connect with FilmFlux on various platforms
- **Mobile-First**: Optimized for all screen sizes

## 📦 Installation

### Prerequisites
- Node.js (version 14 or higher)
- npm or yarn package manager

### Setup Instructions

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/filmflux.git
   cd filmflux
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm start
   ```

4. **Open your browser**
   Navigate to `http://localhost:3000` to view the application

## 🎨 Design System

### Color Palette
- **Primary Green**: `#00ff88` - Brand color for accents and highlights
- **Dark Navy**: `#1a1a2e` - Primary background color
- **Deep Blue**: `#16213e` - Secondary background
- **Light Gray**: `#b8b8b8` - Text color for secondary content

### Typography
- **Poppins**: Used for headings, buttons, and brand elements
- **Inter**: Used for body text, descriptions, and general content

- **Hover Effects**: Scale, glow, and color transitions
- **Page Transitions**: Smooth fade-in animations
- **Micro-interactions**: Button clicks, form submissions
- **Loading States**: Skeleton screens and placeholders

## 🚀 Deployment

### Build for Production
```bash
npm run build
```

### Deploy to Netlify
1. Connect your GitHub repository to Netlify
2. Set build command: `npm run build`
3. Set publish directory: `build`
4. Deploy automatically on push to main branch

### Deploy to Vercel
1. Install Vercel CLI: `npm i -g vercel`
2. Run: `vercel` in project directory
3. Follow the prompts to deploy
